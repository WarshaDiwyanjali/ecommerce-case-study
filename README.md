# E-Commerce Customer Behavior — Data Analytics Case Study

---

## Project Overview

This project analyzes user behavior data from an e-commerce platform to understand customer journey patterns and improve conversion performance.

The main objective is to transform raw event-level data into actionable business insights by examining:

Customer drop-off points in the purchase funnel
Revenue contribution by products, brands, and categories
User behavioral segmentation
Key opportunities to improve conversion rate and revenue

The final output is a structured analytical report supported by KPIs, visualizations, and business recommendations.

## Dataset Description

File: events.csv
Scale: ~885,000 rows × 9 columns

| Column        | Type        | Description                |
| ------------- | ----------- | -------------------------- |
| event_time    | datetime    | Timestamp of user action   |
| event_type    | categorical | view, cart, purchase       |
| product_id    | string      | Unique product identifier  |
| category_id   | string      | Product category ID        |
| category_code | string      | Hierarchical category name |
| brand         | string      | Product brand              |
| price         | float       | Product price              |
| user_id       | string      | Unique user identifier     |
| user_session  | string      | Session identifier         |

## Funnel Analysis

Users who viewed products: 406,863
↓
Users who added to cart: 36,952
↓
Users who completed purchase: 21,304


### Key Insight

The largest drop-off occurs between view and cart stages, making it the highest-impact optimization opportunity.
---

## Key Insights

Most users drop off before adding items to cart
Revenue is concentrated in a small number of users
A small number of products generate most revenue
Improving product page quality is likely to improve conversion rates 

---

## Business Recommendations

Improve product page design, clarity, and trust signals
Focus marketing and inventory on high-performing products and categories
Implement cart abandonment recovery strategies
Improve product metadata quality for low-conversion items
Strengthen customer retention and repeat purchase strategies
